Write the HDL for, and test, the following chips in this order:

Bit (folder a) [X]
Register (folder a) [x]
RAM8 (folder a) [x]
RAM64 (folder a) [x]
RAM512 (folder b)
RAM4K (folder b)
RAM16K (folder b)
PC (folder a)

Put the HDL files only into a and b folders, put them into a folder with 
your last name, first name and project number on it (example: SmithJohnProject3), 
compress the folder and upload.
